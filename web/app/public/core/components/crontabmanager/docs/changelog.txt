1.1.3-pl
==============
- Добавлены две функции $this->timerStart() - время начало запуска и $this->timerEnd() время завершения

1.1.2-pl
==============
- убран лишний элемент

1.1.0-pl
==============
- Добавлен метод для переключение на throw new Exception() в место exit;
- Добавлена проверка запуска из веб браузера или из консоли, для замены занаков переноса: br или PHP_EOL
- Добавлена проверка на повторное объявление класса SchedulerService
- Добавлена обработка чанков с помощью pdoTools пряма в контроллере можно вызвать $this->getChunk()
- Добавлена возможность отправки сообщений из контроллера
    $this->sendMessage(['email@email.ru'],'subject','body')
    или сразу с прекреплением файлов $this->sendMessage(['email@email.ru'],'subject','body',['/full_path/file.zip'])
- Добавлена возможность запаковки содержимого с помощью функции $this->packFile($source, $target);
- Добавление своего сообщения для письма уведомления
- Отчищает пробелы по бокам при поиске задания
