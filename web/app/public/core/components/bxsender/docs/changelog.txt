- Исправлен баг отправки сообщения в ручную

    - bxOnAfterActionClicks событие после записи перехода по ссылке из песьма
    - bxOnBeforeActionClicksRedirect - событие перед рекдиректом указанной в ссылке

- Автоматическая замена двойных урл в письме
